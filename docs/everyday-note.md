# 每天都要写笔记

原因：

1. 产出效率低。经常是这开一篇新笔记，那开一篇新笔记；这篇写一点，那篇写一点，不够专注。
2. 构筑信心。每天发表笔记，形成固定的成就感。
3. 可视化量变积聚。

对策：

- 每天发表 <span style="color:red;">至少一篇</span> 笔记初版。
